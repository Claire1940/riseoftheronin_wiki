<Section icon={Shield} title="System-Level Windows and GPU Tweaks">

System-level configurations often dictate how resources are allocated to the game executable. If Windows or the GPU control panel is not configured correctly, the system may limit power delivery or fail to prioritize the game process, leading to micro-stutters and input lag. Adjusting these settings ensures your hardware operates at its full potential.

<Steps>
  <Step num="1" title="Enable Hardware-Accelerated GPU Scheduling">
    Open the Windows Start menu, search for Graphics Settings, and toggle the Hardware-Accelerated GPU Scheduling (HAGS) option to On. Restart your computer to apply the changes.
  </Step>
  <Step num="2" title="Adjust GPU Power Management">
    Open the Nvidia Control Panel or AMD Radeon Software, navigate to global settings, and set the Power Management Mode to Prefer Maximum Performance.
  </Step>
  <Step num="3" title="Optimize Shader Cache Size">
    In your GPU control panel, increase the Shader Cache Size to 10GB or Unlimited. This allows the system to store compiled shaders, reducing stuttering during gameplay.
  </Step>
  <Step num="4" title="Disable Background Overlays">
    Turn off in-game overlays in applications like Discord, Steam, and GeForce Experience, as these programs can consume system resources and cause performance conflicts.
  </Step>
</Steps>

These system tweaks work in tandem with the in-game settings to create a more stable environment for the game engine. By increasing shader cache limits, you allow the game to retrieve pre-compiled assets instantly, avoiding the sudden drops in performance that occur when entering new zones or initiating combat encounters.
