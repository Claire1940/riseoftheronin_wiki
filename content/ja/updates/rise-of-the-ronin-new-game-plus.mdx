<Section icon={Target} title="難易度「暗夜」とワールド進行">

**Rise of the Roninの二周目**の最大の特徴は、難易度「暗夜」の設定です。これは単なる「ハードモード」ではなく、ゲームのチャレンジレベルを完全に再構築したものです。長屋の留魂録からアクセスできる「暗夜」では、装備、レベル、スキルをすべて保持したままワールドの状態がリセットされます。

「暗夜」モードでは、敵のレベルの最低ラインが60から始まります。キャンペーンをレベル45や50でクリアしたとしても、横浜での最初の遭遇戦では大幅にレベルが不足していると感じるでしょう。進行は地域ごとに制限されており、横浜から江戸、そして最終的に京都へと移動するには、特定の浪人ミッションをクリアする必要があります。

| 特徴 | 黄昏（標準） | 暗夜（二周目） |
| :--- | :--- | :--- |
| **敵レベル** | ストーリーに応じて変化 | レベル60からレベル100以上 |
| **ドロップのレアリティ** | 名品（金）まで | 極上品（緑）の解放 |
| **敵の攻撃性** | 標準的なAI | 攻撃が速く、隙が少ない |
| **治安** | クリア後は固定 | 報酬とともに再出現 |
| **お尋ね者** | クリア後は固定 | 極上品をドロップして再出現 |

  <Step num="3" title="江戸と京都を解放">

  </Step>
</Steps>

<Section icon={Sparkles} title="極上品装備と強化・鍛錬">

極上品装備は、**Rise of the Roninの二周目**における最高峰の装備です。特徴的な緑色のアイコンで識別される極上品は、名品（金）アイテムと比較して基本ステータスが高く、特殊効果のスロット数も多くなっています。これらのアイテムは、難易度「暗夜」、または高レベルの因縁ミッションや隠しボスとの再戦の報酬としてのみドロップします。

クリア後の長屋も大幅にアップグレードされます。拠点から直接すべての鍛冶・製作サービスにアクセスできるようになり、街の特定の商人の元へ行く必要がなくなります。これにより、因縁継承や入手した極上品装備の強化プロセスがスムーズになります。

| 装備の側面 | 名品（金） | 極上品（緑） |
| :--- | :--- | :--- |
| **基本攻撃力/防御力** | 高い | 最大 |
| **特殊効果スロット** | 3-4スロット | 4-5スロット |
| **強化の可能性** | 標準素材で制限あり | 極上品素材が必要 |
| **入手先** | すべての難易度 | 難易度「暗夜」のみ |

<Section icon={Swords} title="能力開花・突破と戦闘の極意">

標準的なスキルツリー（武勇、技能、知略、魅力）を超えて、**Rise of the Roninの二周目**では「能力開花・突破」が導入されます。これらはステータスメニューにあり、「隠し刀」の紋章で示されています。能力開花・突破では、キャラクターの主要属性を無限に（あるいは非常に高い上限まで）成長させることができます。

これらのスキルは、レベル60以上の敵に打ち勝つために必要なパワーを提供するため、エンドゲームを生き抜くために不可欠です。レベル上限に達した状態で経験値を獲得したり、「暗夜」限定の目的を達成したりすることで、能力開花ポイントを獲得できます。

| スキルカテゴリ | 主なメリット | スケーリングタイプ |
| :--- | :--- | :--- |
| **体力開花** | 最大HPの増加 | レベルごとの固定ボーナス |
| **武器開花** | 攻撃力の増加 | パーセントベース |
| **副武器スキル** | 弓/銃/手裏剣の強化 | ダメージとリロード速度 |
| **戦闘スタイル** | 新しい流派の派生を解放 | 独自の技セット |

<Section icon={Trophy} title="リプレイ要素と隠しボス">

留魂録は単に難易度を変更するためだけのものではありません。ストーリーの「もしも」のシナリオを探索するための鍵でもあります。**Rise of the Roninの二周目**では、重要な局面に戻って反対の勢力（佐幕派 vs 倒幕派）を選択し、世界がどのように変化するかを確認したり、以前はアクセスできなかった新しいミッションを解放したりできます。

さらに、世界にはマップの辺境に隠された「隠しボス」——強力な武芸者たちが存在します。「暗夜」モードでは、これらのボスはレベル100相当に調整されており、ゲーム内で最も強力な極上品アイテムをドロップします。

| アクティビティ | 報酬タイプ | リプレイ価値 |
| :--- | :--- | :--- |
| **勢力の切り替え** | 新しいストーリーカット/ミッション | 高（全コンテンツの確認） |
| **隠しボス** | 固有の極上品セット | 高（最強装備） |
| **因縁ミッション** | 同行者の極上品装備 | 中（ビルド依存） |
| **治安** | 消耗品および素材 | 低（周回用） |

<Callout type="warning" title="歴史の整合性">
留魂録で選択肢を変更すると、それが現在のワールドの状態に「反映」されます。両方のルートの報酬を保持したい場合は、ミッションを完了して新しいタイムラインを保存する必要があります。
</Callout>

</Section>
