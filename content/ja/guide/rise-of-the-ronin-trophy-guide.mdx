### 流派の有効性

敵の武器に対して自分の流派（天・地・人）を合わせることは、気力を削り、戦闘関連のトロフィーを獲得する上で非常に重要です。

| 流派タイプ | 有利な相手 | 不利な相手 |
| :--- | :--- | :--- |
| **天** | 刀 / 中量武器 | 剣 / 軽量武器 |
| **地** | 大太刀 / 重量武器 | 刀 / 中量武器 |
| **人** | 剣 / 軽量武器 | 大太刀 / 重量武器 |
| **忍** | 特殊 / 混合 | 標準的な相性 |

<Callout type="success" title="ビルドの提案">
トロフィー「暗夜を越えて」のために、**魅力（Charisma）ベースの気力削りビルド**に焦点を当てましょう。これにより、石火の後に気力を急速に回復でき、強力なボス戦が非常に楽になります。
</Callout>

### 人間関係の報酬
仲間との因縁を深めることで、**「比翼の契り」**（恋愛関係）トロフィーが解放されます。少なくとも1人の対象キャラクターと因縁レベルを最大にする必要があります。

| アクティビティ | 因縁への影響 | トロフィーの進行 |
| :--- | :--- | :--- |
| **贈り物を渡す** | 上昇（大） | 比翼の契りへの到達を早めます。 |
| **因縁ミッション** | 上昇（特大） | ユニークな武器や流派を解放します。 |
| **長屋での会話** | 上昇（中） | 背景の物語を知ることができ、僅かなブーストを得られます。 |
| **徒党ミッション** | 上昇（大） | トロフィー「一期一会」に必要です。 |

<Callout type="info" title="備考">

</Callout>

<Section icon={Shield} title="エンドゲーム：難易度「暗夜」とクリーンアップ">

**Rise of the Ronin トロフィーガイド**の最後のハードルは、クリア後のコンテンツです。メインストーリーを完了すると、難易度「暗夜」が解放されます。このモードでは敵がより強力になり、より良い戦利品がドロップします。

1. **暗夜を越えて**: 難易度「暗夜」で任意のミッションをクリアします。最も簡単な方法は、エンドゲームの装備を使用してプロローグの最初のミッションを再プレイすることです。
2. **レベルの極み**: レベル55に到達します。すべての地域をクリアしていれば、これに近いレベルになっているはずです。足りない場合は、「暗夜」でミッションを再プレイすると大量の経験値が得られます。
3. **装備の強化**: 装備を最大レベルまで強化します。クリア後に入手できる「極上品」（赤色）のアイテムのために素材を温存しておきましょう。

<Callout type="warning" title="警告">
装備の強化には多額の費用がかかります。戦闘効率を最大化するために、4つの互換性のある揃え効果を持つ武器に絞って投資することをお勧めします。
</Callout>
