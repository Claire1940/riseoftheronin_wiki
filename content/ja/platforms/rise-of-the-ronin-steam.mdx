<Section icon={Shield} title="ハードウェア要件と目標パフォーマンス">

PC版 **Rise of the Ronin** は、Team NINJAによる幕末を舞台としたアクション重視のオープンワールドRPGをPC環境に提供します。横浜、江戸、京都を駆け巡る流麗な剣戟と高速移動を体験するには、システムが公式のハードウェア基準を満たしている必要があります。PC版ではDirectX 12への対応と、シームレスなワールド読み込みのための堅牢なマルチチャネルメモリ構成が求められます。

| 仕様 | 最低スペック | 推奨スペック |
| :--- | :--- | :--- |
| OS | Windows 10/11 (64ビット) | Windows 10/11 (64ビット) |
| プロセッサー | Intel i5-10400 / AMD Ryzen 5 1600 | Intel i5-10600K / AMD Ryzen 5 5600X |
| メモリ | 16 GB RAM | 16 GB RAM |
| グラフィックボード | NVIDIA GTX 1060 / AMD RX 5500 XT | NVIDIA RTX 2080 Super / AMD RX 6700 XT |
| ストレージ | 155 GB SSD 必須 | 155 GB NVMe SSD 推奨 |
| DirectX | バージョン 12 | バージョン 12 |

最低構成は、最低グラフィックプリセットとアップスケーリングを組み合わせて1080p解像度、30 FPSでの動作を目標としています。推奨構成は、標準的な設定で1080p/60 FPSのよりスムーズな動作を目指しています。都市部などの密集地を移動する際のマイクロスタッタリングを防ぐため、マルチチャネルメモリ構成が強く推奨されます。

<Callout type="warning" title="SSD必須">
ゲームの実行にはSSDが厳格に要求されます。HDDにインストールすると、テクスチャのポップイン、読み込み中のフリーズ、カットシーンでの音声のズレが深刻なレベルで発生します。
</Callout>

<Section icon={Sparkles} title="Steam版グラフィック設定と最適化">

PC版には、ハイエンドハードウェアを活用するための最新の表示機能が多数搭載されています。フレームレートを最大120 FPSまで引き上げたり、最大8Kの解像度で出力したりすることが可能です。また、ウルトラワイドおよびスーパーウルトラワイドモニターへのネイティブ対応により、19世紀日本の映画のような風景を存分に楽しめます。

| 機能 | 対応レベル | 技術詳細 |
| :--- | :--- | :--- |
| 最大解像度 | 最大 8K Ultra HD | 対応するハイエンドディスプレイで利用可能 |
| 目標フレームレート | 最大 120 FPS | 高リフレッシュレートモニターへのネイティブ対応 |
| アップスケーリング技術 | DLSS, FSR, XeSS | パフォーマンス向上のため複数のベンダーオプションを用意 |
| フレーム生成 | 対応 | 対応GPUで視覚的な滑らかさを向上 |
| レイトレーシング | 対応 | ライティング、影、反射の表現を強化 |
| ウルトラワイド対応 | 対応 | 21:9 および 32:9 のアスペクト比にネイティブ対応 |

安定した60 FPS以上を実現するには、個々の設定を慎重に調整する必要があります。影の品質、レイトレーシング、ボリュームレンダリングは、ハードウェアのパフォーマンスに最も大きな影響を与えます。

| 設定項目 | 推奨設定 | パフォーマンスへの影響 |
| :--- | :--- | :--- |
| 描画解像度 | ネイティブ / DLSS クオリティ | GPU負荷に大きく影響 |
| レイトレーシング | オフ / 低 | 有効にするとパフォーマンスが大幅に低下 |
| 影の品質 | 中 | CPUおよびGPUを中程度に使用 |
| アンビエントオクルージョン | 標準 | VRAM使用量を抑えつつ奥行きを改善 |
| テクスチャ品質 | 高 (8GB+ VRAM) | VRAM消費は多いがFPSへの影響は最小限 |
| モーションブラー | オフ | 視覚的な好みによるが、わずかにFPS向上 |

<Callout type="tip" title="フレーム生成のアドバイス">
ミドルレンジのグラフィックボードで安定した60 FPSを維持できない場合は、DLSSまたはFSRを「クオリティ」モードに設定し、フレーム生成を有効にすることで、視覚的な鮮明さを損なわずにパフォーマンスを大幅に向上させることができます。
</Callout>

</Section>

<Section icon={Map} title="Steam協力プレイの設定とオンラインマッチングガイド">

PC版では、最大3人までのシームレスなオンライン協力プレイに対応しています。マッチングはSteamのネットワークインフラを直接使用するため、オンライン機能を利用するための外部サブスクリプションサービスは不要です。

| 機能の種類 | 仕組み | ゲームプレイへの影響 |
| :--- | :--- | :--- |
| 直接協力プレイ | 特定のミッションで最大3人を募集または参加 | 共闘による戦闘、ボス戦の協力 |
| 合言葉マッチング | キーを設定してフレンドのみとロビーを共有 | 野良マッチングなしのプライベートセッション |
| 非同期浪人 | 他プレイヤーの分身がNPCとしてワールドに登場 | 倒して戦利品を得るか、味方として雇う |
| こんぴら狗 | お金を預けて他プレイヤーのワールドへ犬を送り出す | 後に希少なアイテムや素材を持って帰還 |

セッションをホストするには、対応ミッションの協力プレイ準備画面で「募集」を選択します。特定のフレンドと遊びたい場合は、両者がマッチング設定で全く同じ「合言葉」を設定していることを確認してください。

<Callout type="info" title="協力プレイの制限">
協力プレイはミッション単位です。フレンドとオープンワールドを自由に探索することはできません。ミッションが完了すると、各プレイヤーはそれぞれのソロワールドに戻ります。
</Callout>

</Section>
